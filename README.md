## About:

This Library was created to easily share code between The Panda Oliver's mods.

### Current Features:

* Config API 
  * Automatic Config synchronization

### Future Features
* **In development**
  * Multi-Loader Framework (v1.0.0)
  * Custom Model Rendering API


* **Planned**
  * Config Menu API
  * Wiki / Docs
